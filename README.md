Topic_to_List.py : 원하는 토픽 입력 시 토픽에 대한 목차를 자동으로 생성해주는 툴 만들기
기능
- 사용자가 원하는 토픽(주제)를 입력받음
- 입력받은 토픽(주제)에 대한 목차를 생성해줌
- 만약 생성된 목차를 수정하고 싶은 경우 원하는 항목을 선택하여 수정 가능
- 프롬프트의 내용을 해당 토픽(주제) 의 파일명을 가진 txt 파일 저장 가능
